Documentation notebook functions¶

from fastai.gen_doc.nbdoc import *
from fastai.gen_doc.nbdoc import *

nbdoc contains the functions for documentation notebooks. The most important is show_doc:

show_doc(show_doc)

show_doc

(elt, doc_string:bool=True, full_name:str=None, arg_comments:dict=None, title_level=None, alt_doc_string:str='', ignore_warn:bool=False)

[source]

Show documentation for element elt. Supported types: class, Callable, and enum

In your notebook you'll have a call to show_doc for every symbol you wish to document. And you'll want to hide input for that cell. For instance, the above signature is generated by this function:

show_doc(show_doc)
